获取所有可用的技术指标定义，包括指标名称、描述和返回字段。

<CodeGroup>

```bash cURL
curl https://api.reportify.cn/v1/quant/indicators \
  -H "Authorization: Bearer <token>"
```

```python Python
import requests

response = requests.get(
    "https://api.reportify.cn/v1/quant/indicators",
    headers={"Authorization": "Bearer <token>"}
)
print(response.json())
```

```typescript TypeScript
const response = await fetch('https://api.reportify.cn/v1/quant/indicators', {
  headers: {
    'Authorization': 'Bearer <token>'
  }
});
const data = await response.json();
```

</CodeGroup>

## 响应参数

返回 `IndicatorMeta[]` 数组，指标列表。

<Expandable title="IndicatorMeta 字段">
  <ResponseField name="name" type="string">
    指标名称，用于公式中（如 MACD, RSI, KDJ）
  </ResponseField>
  <ResponseField name="description" type="string">
    指标描述
  </ResponseField>
  <ResponseField name="fields" type="string[]">
    返回字段名列表（如 MACD 返回 `["dif", "dea", "macd"]`）
  </ResponseField>
</Expandable>

## 响应示例

```json
[
  {
    "name": "RSI",
    "description": "Relative Strength Index",
    "fields": ["rsi"]
  },
  {
    "name": "MACD",
    "description": "Moving Average Convergence Divergence",
    "fields": ["dif", "dea", "macd"]
  },
  {
    "name": "KDJ",
    "description": "Stochastic Oscillator",
    "fields": ["k", "d", "j"]
  },
  {
    "name": "BOLL",
    "description": "Bollinger Bands",
    "fields": ["upper", "mid", "lower"]
  }
]
```

## 指标分类

### Level 0 - 核心函数

通达信风格：列参数在前，周期参数在后。

| 类别 | 函数 | 示例 | 说明 |
|------|------|------|------|
| 数学运算 | `ABS(X)`, `LN(X)`, `POW(X, N)`, `SQRT(X)`, `MAX(X, Y)`, `MIN(X, Y)` | `ABS(CLOSE - OPEN)` | 基础数学函数 |
| 移动平均 | `MA(X, N)`, `EMA(X, N)`, `SMA(X, N, M)`, `DMA(X, A)`, `WMA(X, N)` | `MA(CLOSE, 20)` | 各类均线 |
| 统计函数 | `HHV(X, N)`, `LLV(X, N)`, `STD(X, N)`, `AVEDEV(X, N)`, `SUM(X, N)`, `REF(X, N)` | `HHV(HIGH, 20)` | 统计计算 |
| 逻辑函数 | `IF(C, A, B)`, `CONST(X)`, `DIFF(X, N)`, `RD(X, N)`, `RET(X, N)` | `IF(CLOSE > OPEN, 1, 0)` | 条件判断 |

### Level 1 - 应用函数

| 类别 | 函数 | 示例 | 说明 |
|------|------|------|------|
| 交叉信号 | `CROSS(A, B)`, `CROSSDOWN(A, B)`, `LONGCROSS(A, B, N)` | `CROSS(MA(CLOSE, 5), MA(CLOSE, 20))` | 均线交叉判断 |
| 计数函数 | `COUNT(X, N)`, `EVERY(X, N)`, `EXIST(X, N)`, `BARSLAST(X)`, `LAST(X, A, B)` | `COUNT(CLOSE > OPEN, 10)` | 条件计数 |
| 取值函数 | `VALUEWHEN(C, X)`, `BETWEEN(X, A, B)` | `VALUEWHEN(CROSS(MA(CLOSE, 5), MA(CLOSE, 20)), CLOSE)` | 条件取值 |

### Level 2 - 趋势指标

| 指标 | 公式 | 返回字段 | 描述 |
|------|------|----------|------|
| MACD | `MACD(S, L, M)` | `dif`, `dea`, `macd` | 指数平滑异同移动平均 |
| BOLL | `BOLL(N, P)` | `upper`, `mid`, `lower` | 布林带 |
| TAQ | `TAQ(N)` | - | 唐奇安通道 |
| KTN | `KTN(N, M)` | - | 肯特纳通道 |
| EXPMA | `EXPMA(N)` | - | 指数移动平均 |
| BBI | `BBI()` | - | 多空指数 |

### Level 2 - 动量指标

| 指标 | 公式 | 返回字段 | 描述 |
|------|------|----------|------|
| KDJ | `KDJ(N, M1, M2)` | `k`, `d`, `j` | 随机指标 |
| RSI | `RSI(N)` | `rsi` | 相对强弱指数 |
| WR | `WR(N)` | `wr` | 威廉指标 |
| CCI | `CCI(N)` | `cci` | 商品通道指数 |
| BIAS | `BIAS(N)` | `bias` | 乖离率 |
| PSY | `PSY(N)` | `psy` | 心理线 |
| MTM | `MTM(N)` | `mtm` | 动量指标 |
| ROC | `ROC(N)` | `roc` | 变动率 |

### Level 2 - 波动率指标

| 指标 | 公式 | 返回字段 | 描述 |
|------|------|----------|------|
| ATR | `ATR(N)` | `atr` | 平均真实波幅 |
| DMI | `DMI(N, M)` | `pdi`, `mdi`, `adx`, `adxr` | 趋向指标 |
| TRIX | `TRIX(N)` | `trix` | 三重指数平滑 |
| VR | `VR(N)` | `vr` | 成交量变异率 |
| CR | `CR(N)` | `cr` | 能量指标 |
| EMV | `EMV(N)` | `emv` | 简易波动 |
| DPO | `DPO(N)` | `dpo` | 去趋势价格 |
| BRAR | `BRAR(N)` | `br`, `ar` | 情绪指标 |
| OBV | `OBV()` | `obv` | 能量潮 |
| MFI | `MFI(N)` | `mfi` | 资金流量指标 |
| ASI | `ASI(N)` | `asi` | 累积摆动指标 |
