获取单只股票的分时数据。

<CodeGroup>

```bash cURL
curl "https://api.reportify.cn/v1/quant/quotes/minute?symbol=000001&start_datetime=2026-04-01%2009:30:00&end_datetime=2026-04-01%2015:00:00&market=cn" \
  -H "Authorization: Bearer <token>"
```

```python Python
import requests

response = requests.get(
    "https://api.reportify.cn/v1/quant/quotes/minute",
    headers={"Authorization": "Bearer <token>"},
    params={
        "symbol": "000001",
        "start_datetime": "2026-04-01 09:30:00",
        "end_datetime": "2026-04-01 15:00:00",
        "market": "cn"
    }
)
print(response.json())
```

```typescript TypeScript

  method: 'GET',
```

## 使用示例

### 获取当日交易数据

```python
import requests
from datetime import datetime

today = datetime.now().strftime("%Y-%m-%d")
response = requests.get(
    "https://api.reportify.cn/v1/quant/quotes/minute",
    headers={"Authorization": "Bearer <token>"},
    params={
        "symbol": "600519",
        "start_datetime": f"{today} 09:30:00",
        "end_datetime": f"{today} 15:00:00",
        "market": "cn"
    }
)
data = response.json()

for row in data["datas"][:10]:
    print(f"{row['date']}: 价{row['current']} 量{row['volume']}")
```
