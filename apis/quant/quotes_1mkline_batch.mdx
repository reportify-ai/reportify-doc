批量获取多只股票的 1 分钟 K 线（开高低收量）数据。

数据按日期（升序）和股票代码排序。

<CodeGroup>

```bash cURL
curl -X POST https://api.reportify.cn/v1/quant/quotes/1mkline/batch \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "market": "cn",
    "symbols": ["000001", "000002", "600519"],
    "start_datetime": "2024-01-01 09:30:00",
    "end_datetime": "2024-01-01 15:00:00"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.reportify.cn/v1/quant/quotes/1mkline/batch",
    headers={"Authorization": "Bearer <token>"},
    json={

        "start_datetime": "2024-01-01 09:30:00",
        "end_datetime": "2024-01-01 15:00:00"
    }
)
print(response.json())
```

<ParamField body="market" type="string" default="cn">

</ParamField>

<ParamField body="start_datetime" type="string" required>

<ParamField body="end_datetime" type="string" required>

## 使用示例

### 批量获取当日交易数据

```python
import requests
from datetime import datetime

# 获取当日 9:30 到 15:00 的 1 分钟 K 线
today = datetime.now().strftime("%Y-%m-%d")
response = requests.post(
    "https://api.reportify.cn/v1/quant/quotes/1mkline/batch",
    headers={"Authorization": "Bearer <token>"},
    json={
        "market": "cn",
        "symbols": ["000001", "000002", "600519"],
        "start_datetime": f"{today} 09:30:00",
        "end_datetime": f"{today} 15:00:00"
    }
)
data = response.json()

# 按股票分组
from collections import defaultdict
by_symbol = defaultdict(list)
for row in data["datas"]:
    by_symbol[row["symbol"]].append(row)

# 输出每只股票的最新数据
for symbol, rows in by_symbol.items():
    latest = rows[-1]
    print(f"{symbol}: {latest['datetime']} 收{latest['close']} 量{latest['volume']}")
```
