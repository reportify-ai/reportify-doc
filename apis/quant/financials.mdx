title: '获取财务数据'
description: '获取股票历史财务报表数据'
openapi: 'GET /v1/quant/financials/{financial_type}'
---

获取单只股票的历史财务报表数据，支持利润表、现金流量表、资产负债表和股东权益变动表。

<CodeGroup>

```bash cURL
curl "https://api.reportify.cn/v1/quant/financials/income?symbol=000001&market=cn&start_year=2023&end_year=2025" \
  -H "Authorization: Bearer <token>"
```

```python Python
import requests

response = requests.get(
    "https://api.reportify.cn/v1/quant/financials/income",
    headers={"Authorization": "Bearer <token>"},
    params={
        "symbol": "000001",
        "market": "cn",
        "start_year": 2023,
        "end_year": 2025
    }
)
print(response.json())
```

```typescript TypeScript
const params = new URLSearchParams({
  symbol: '000001',
  market: 'cn',
  start_year: '2023',
  end_year: '2025'
});

```

</CodeGroup>

## 请求参数

<ParamField path="financial_type" type="string" required>
  财务报表类型：
  - `income` - 利润表（营收、利润等）
  - `cashflow` - 现金流量表
  - `balancesheet` - 资产负债表
  - `equity` - 股东权益变动表
</ParamField>

<ParamField query="symbol" type="string" required>
  股票代码
</ParamField>

<ParamField query="market" type="string" default="cn">
  股票市场：`cn`（A股）, `hk`（港股）, `us`（美股）
</ParamField>

<ParamField query="start_year" type="integer">
  开始年份（默认：3 年前）
</ParamField>

<ParamField query="end_year" type="integer">
  结束年份（默认：当前年份）
</ParamField>

## 响应参数

<ResponseField name="datas" type="array">
  财务数据列表，按报告期排序
  <Expandable title="数据字段">
    <ResponseField name="symbol" type="string">
      股票代码
    </ResponseField>
    <ResponseField name="report_period" type="string">
      报告期（如：2024-12-31）
    </ResponseField>
    <ResponseField name="..." type="various">
      财务指标字段（根据报表类型不同）
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="metadata" type="object">
  查询元数据
</ResponseField>

## 财务报表类型

### 利润表 (income)

包含营业收入、营业成本、利润等指标：

| 字段 | 说明 |
|------|------|
| `revenue` | 营业收入 |
| `operating_cost` | 营业成本 |
| `gross_profit` | 毛利润 |
| `operating_profit` | 营业利润 |
| `net_profit` | 净利润 |
| `eps` | 每股收益 |

### 现金流量表 (cashflow)

包含经营、投资、筹资活动现金流：

| 字段 | 说明 |
|------|------|
| `operating_cashflow` | 经营活动现金流量净额 |
| `investing_cashflow` | 投资活动现金流量净额 |
| `financing_cashflow` | 筹资活动现金流量净额 |
| `net_cashflow` | 现金及现金等价物净增加额 |

### 资产负债表 (balancesheet)

包含资产、负债、所有者权益：

| 字段 | 说明 |
|------|------|
| `total_assets` | 总资产 |
| `total_liabilities` | 总负债 |
| `total_equity` | 所有者权益 |
| `current_assets` | 流动资产 |
| `current_liabilities` | 流动负债 |

### 股东权益变动表 (equity)

包含股东权益变动明细：

| 字段 | 说明 |
|------|------|
| `share_capital` | 股本 |
| `capital_reserve` | 资本公积 |
| `retained_earnings` | 留存收益 |
| `minority_interest` | 少数股东权益 |

## 使用示例

### 获取利润表

```python
import requests

response = requests.get(
    "https://api.reportify.cn/v1/quant/financials/income",
    headers={"Authorization": "Bearer <token>"},
    params={"symbol": "600519", "market": "cn"}
)
data = response.json()

for row in data["datas"]:
    print(f"{row['report_period']}: 营收{row['revenue']/1e8:.2f}亿 净利润{row['net_profit']/1e8:.2f}亿")
```

### 获取资产负债表

```python
response = requests.get(

        "market": "cn",
        "start_year": 2022,
        "end_year": 2025
    }
)
```
