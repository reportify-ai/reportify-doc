## 获取技术指标

获取指定股票的技术指标数据。

<CodeGroup>

```bash cURL
curl -X POST https://api.reportify.cn/v2/quant/indicators \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "symbols": ["US:AAPL", "US:MSFT"],
    "indicators": ["ma", "rsi", "macd"],
    "start_date": "2024-01-01",
    "end_date": "2024-06-30",
    "interval": "1d",
    "params": {
      "ma_period": 20,
      "rsi_period": 14
    }
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.reportify.cn/v2/quant/indicators",
    headers={"Authorization": "Bearer <token>"},
    json={
        "symbols": ["US:AAPL", "US:MSFT"],
        "indicators": ["ma", "rsi", "macd"],
        "start_date": "2024-01-01",
        "params": {"ma_period": 20, "rsi_period": 14}
    }
)
print(response.json())
```

```typescript TypeScript

```

</CodeGroup>

### 请求参数

<ParamField body="symbols" type="array" required>
  股票代码列表，格式：`市场:代码`
</ParamField>

<ParamField body="indicators" type="array" required>
  指标名称列表，可选值：`ma`, `ema`, `rsi`, `macd`, `bollinger`, `atr`, `obv`, `kdj`
</ParamField>

<ParamField body="start_date" type="string">
  开始日期，格式：`YYYY-MM-DD`
</ParamField>

<ParamField body="end_date" type="string">
  结束日期，格式：`YYYY-MM-DD`
</ParamField>

<ParamField body="interval" type="string" default="1d">
  数据周期：`1d`（日线）, `1w`（周线）, `1m`（月线）
</ParamField>

<ParamField body="params" type="object">
  指标参数配置
  
  <Expandable title="常用参数">
    - `ma_period`: 移动平均周期（默认 20）
    - `ema_period`: EMA 周期（默认 12）
    - `rsi_period`: RSI 周期（默认 14）
    - `macd_fast`: MACD 快线周期（默认 12）
    - `macd_slow`: MACD 慢线周期（默认 26）
    - `macd_signal`: MACD 信号线周期（默认 9）
    - `bollinger_period`: 布林带周期（默认 20）
    - `bollinger_std`: 布林带标准差倍数（默认 2）
  </Expandable>
</ParamField>

### 响应参数

<ResponseField name="status" type="integer">
  HTTP 状态码
</ResponseField>

<ResponseField name="code" type="integer">
  响应代码（0 表示成功）
</ResponseField>

<ResponseField name="message" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="array">
  指标数据列表
  
  <Expandable title="数据字段">
    - `date`: 日期
    - `symbol`: 股票代码
    - `open`: 开盘价
    - `high`: 最高价
    - `low`: 最低价
    - `close`: 收盘价
    - `volume`: 成交量
    - `ma_20`: 20日移动平均（如果请求）
    - `rsi_14`: 14日 RSI（如果请求）
    - `macd`: MACD 值（如果请求）
    - `macd_signal`: MACD 信号线（如果请求）
    - `macd_histogram`: MACD 柱状图（如果请求）
  </Expandable>
</ResponseField>

## 获取指标列表

获取所有可用的技术指标定义。

```bash

```
