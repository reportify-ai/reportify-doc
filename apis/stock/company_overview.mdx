## 公司概览

**URL**: `/v1/stock/company-overview`
**方法**: `POST`  
**描述**: 获取公司综合信息，包括基本面和业务概览。

### 请求参数
| 参数名  | 类型   | 必填 | 描述                                      |
|---------|--------|------|-------------------------------------------|
| symbols | str    | 否   | 股票代码，支持多个，用逗号分隔，如 `HK:00700,US:AAPL` |
| names   | str    | 否   | 股票名称，支持多个，用逗号分隔，如 `腾讯,苹果`       |

<Note>
`symbols` 和 `names` 至少需要提供一个参数。
</Note>

| 参数名              | 类型    | 描述                                       |
|---------------------|---------|-------------------------------------------|
| status              | int     | HTTP 状态码。                              |
| code                | int     | 响应代码（0 表示成功）。                   |
| message             | str     | 响应消息。                                 |
| data                | object  | 公司信息数据。                             |
| &nbsp;&nbsp;country                      | str     | 注册国家/地区代码。                        |
| &nbsp;&nbsp;nature_of_company            | str     | 公司性质。                                 |
| &nbsp;&nbsp;fiscal_year_end              | str     | 财年结束日期（MM-DD）。                    |
| &nbsp;&nbsp;city                         | str     | 所在城市。                                 |
| &nbsp;&nbsp;stockList                    | array   | 股票列表信息。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_full_name           | str     | 股票全称。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;market                    | str     | 市场代码。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;symbol                    | str     | 股票代码。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_english_name        | str     | 股票英文名称。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;industry_HS_1             | str     | 行业分类一级。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;industry_HS_2             | str     | 行业分类二级。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_name                | str     | 股票名称。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;asset_class               | str     | 资产类别。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_english_full_name   | str     | 股票英文全称。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;exchange                  | str     | 交易所代码。                               |
| &nbsp;&nbsp;&nbsp;&nbsp;industry_HS_3             | str     | 行业分类三级。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;ipo_date                  | str     | IPO 日期（YYYY-MM-DD）。                   |
| &nbsp;&nbsp;company_english_full_name    | str     | 公司英文全称。                             |
| &nbsp;&nbsp;province                     | str     | 省份/地区。                                |
| &nbsp;&nbsp;fax                          | str     | 传真号码。                                 |
| &nbsp;&nbsp;email                        | str     | 联系邮箱。                                 |
| &nbsp;&nbsp;website                      | str     | 公司网站。                                 |
| &nbsp;&nbsp;address                      | str     | 注册地址。                                 |
| &nbsp;&nbsp;company_english_name         | str     | 公司英文名称。                             |
| &nbsp;&nbsp;telephone                    | str     | 联系电话。                                 |
| &nbsp;&nbsp;registration_date            | str     | 注册日期（YYYY-MM-DD）。                   |
| &nbsp;&nbsp;office_address               | str     | 办公地址。                                 |
| &nbsp;&nbsp;board_secretary              | str     | 董事会秘书。                               |
| &nbsp;&nbsp;company_name                 | str     | 公司名称。                                 |
| &nbsp;&nbsp;company_full_name            | str     | 公司全称。                                 |
| &nbsp;&nbsp;main_business                | str     | 主营业务。                                 |

### 请求示例

**cURL**
```bash
curl -X POST https://api.reportify.cn/v1/stock/company-overview \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "symbols": "HK:00700,US:AAPL"
}'
```

**Python**
```python
import requests

url = "https://api.reportify.cn/v1/stock/company-overview"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

# 通过股票代码查询
data = {"symbols": "HK:00700,US:AAPL"}

# 或者通过股票名称查询
# data = {"names": "腾讯,苹果"}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
