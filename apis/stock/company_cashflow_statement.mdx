## 现金流量表

**URL**: `/v1/stock/company-cashflow-statement`
**方法**: `POST`  
**描述**: 获取公司现金流量表数据。

### 请求参数
| 参数名         | 类型   | 必填 | 描述                                                              |
|----------------|--------|------|-------------------------------------------------------------------|
| symbol         | str    | 是   | 股票代码。                                                              |
| period         | str    | 否   | 报告周期：`annual`、`quarterly`、`cumulative quarterly`。         |
| limit          | int    | 否   | 返回最近 N 条记录，默认 8。                                       |
| start_date     | str    | 否   | 开始日期（YYYY-MM-DD）。                                          |
| end_date       | str    | 否   | 结束日期（YYYY-MM-DD）。                                          |
| calendar       | str    | 否   | `calendar` 或 `fiscal`，默认 `fiscal`。                           |
| fiscal_year    | str    | 否   | 指定财年（如 `2023`）。                                           |
| fiscal_quarter | str    | 否   | 指定财季（Q1, Q2, Q3, Q4, FY, H1, `Q3 (9 months)`）。             |

### 响应参数
| 参数名              | 类型    | 描述                                       |
|---------------------|---------|-------------------------------------------|
| status              | int     | HTTP 状态码。                              |
| code                | int     | 响应代码（0 表示成功）。                   |
| message             | str     | 响应消息。                                 |
| data                | array   | 现金流量表数据列表。                       |
| &nbsp;&nbsp;currency                                  | str     | 货币单位。                                 |
| &nbsp;&nbsp;end_date                                  | str     | 报告期结束日期（YYYY-MM-DD）。             |
| &nbsp;&nbsp;symbol                                    | str     | 股票代码。                                 |
| &nbsp;&nbsp;fiscal_year                               | str     | 财年。                                     |
| &nbsp;&nbsp;fiscal_quarter                            | str     | 财季。                                     |
| &nbsp;&nbsp;period                                    | str     | 报告周期类型。                             |
| &nbsp;&nbsp;filling_date                              | str     | 填报日期（YYYY-MM-DD）。                   |
| &nbsp;&nbsp;cash_flows_from_operating_activities     | object  | 经营活动现金流。                           |
| &nbsp;&nbsp;&nbsp;&nbsp;net_cash_flows_from_operating_activities | number | 经营活动产生的现金流量净额。               |
| &nbsp;&nbsp;cash_flows_from_investing_activities     | object  | 投资活动现金流。                           |
| &nbsp;&nbsp;&nbsp;&nbsp;net_cash_flows_from_investing_activities | number | 投资活动产生的现金流量净额。               |
| &nbsp;&nbsp;cash_flows_from_financing_activities     | object  | 筹资活动现金流。                           |
| &nbsp;&nbsp;&nbsp;&nbsp;net_cash_flows_from_financing_activities | number | 筹资活动产生的现金流量净额。               |
| &nbsp;&nbsp;&nbsp;&nbsp;effect_of_exchange_rate_changes_on_cash_and_cash_equivalents | number | 汇率变动对现金及现金等价物的影响。         |
| &nbsp;&nbsp;&nbsp;&nbsp;net_increase_in_cash_and_cash_equivalents | number | 现金及现金等价物净增加额。                 |
| &nbsp;&nbsp;&nbsp;&nbsp;cash_and_cash_equivalents_at_beginning_of_year | number | 期初现金及现金等价物余额。                 |
| &nbsp;&nbsp;&nbsp;&nbsp;cash_and_cash_equivalents_at_end_of_year | number | 期末现金及现金等价物余额。                 |

### 请求示例

**cURL**
```bash
curl -X POST https://api.reportify.cn/v1/stock/company-cashflow-statement \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json" \
-d '{
    "symbol": "00700",
    "limit": 2
}'
```

**Python**
```python
import requests

url = "https://api.reportify.cn/v1/stock/company-cashflow-statement"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}
data = {
    "symbol": "00700",
    "limit": 2
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
