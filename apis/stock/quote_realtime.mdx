## 实时行情

**URL**: `/v1/stock/quote-realtime`
**方法**: `POST`  
**描述**: 获取股票实时行情数据。

### 请求参数
| 参数名  | 类型   | 必填 | 描述                                          |
|---------|--------|------|-----------------------------------------------|
| symbol  | str    | 是   | 股票代码。                                                |

### 响应参数
| 参数名              | 类型    | 描述                                       |
|---------------------|---------|-------------------------------------------|
| status              | int     | HTTP 状态码。                              |
| code                | int     | 响应代码（0 表示成功）。                   |
| message             | str     | 响应消息。                                 |
| data                | object  | 实时行情数据。                             |
| &nbsp;&nbsp;symbol                  | str     | 股票代码。                                 |
| &nbsp;&nbsp;trade_volume            | int     | 成交量。                                   |
| &nbsp;&nbsp;stock_open              | number  | 开盘价。                                   |
| &nbsp;&nbsp;previous_stock_price    | number  | 前收盘价。                                 |
| &nbsp;&nbsp;stock_change_percent    | number  | 涨跌幅（百分比形式）。                     |
| &nbsp;&nbsp;day_high                | number  | 最高价。                                   |
| &nbsp;&nbsp;trade_amount            | number  | 成交额。                                   |
| &nbsp;&nbsp;name                    | str     | 股票名称。                                 |
| &nbsp;&nbsp;day_low                 | number  | 最低价。                                   |
| &nbsp;&nbsp;exchange                | str     | 交易所代码。                               |
| &nbsp;&nbsp;stock_price             | number  | 当前价格。                                 |
| &nbsp;&nbsp;stock_change            | number  | 涨跌额。                                   |
| &nbsp;&nbsp;timestamp               | str     | 时间戳（ISO 8601 格式）。                  |

### 请求示例

**cURL**
```bash
curl -X POST https://api.reportify.cn/v1/stock/quote-realtime \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json" \
-d '{
    "symbol": "00700"
}'
```

**Python**
```python
import requests

url = "https://api.reportify.cn/v1/stock/quote-realtime"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}
data = {
    "symbol": "00700"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
