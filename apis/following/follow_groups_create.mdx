## 创建关注组

**URL**: `/v1/following/groups`  
**方法**: `POST`  
**描述**: 创建一个新的关注组

### 请求参数
| 参数名         | 类型                   | 描述                                                                 |
|----------------|------------------------|----------------------------------------------------------------------|
| name           | str                   | 组名称                                                               |
| follow_values   | list[FollowValue]      | 关注项列表（可选），每个元素包含 `follow_type` 和 `follow_value` |

FollowValue 结构：

| 参数名         | 类型   | 描述                  |
|----------------|--------|-----------------------|
| follow_type    | int    | 关注类型：1=公司，3=频道 |
| follow_value   | str    | 关注值（如股票代码、频道 ID）   |

### 响应参数
| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| group_id       | str   | 关注组 ID             |
| name           | str   | 组名称                 |
| count          | int   | 关注项数量               |
