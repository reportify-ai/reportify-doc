## 获取关注组列表

**URL**: `/v1/following/groups`  
**方法**: `GET`  
**描述**: 获取当前用户的所有关注组

### 请求参数
无需参数

### 响应参数
| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| groups         | list  | 关注组列表               |

每个关注组包含：

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| group_id       | str   | 关注组 ID             |
| name           | str   | 组名称                 |
| count          | int   | 关注项数量               |
