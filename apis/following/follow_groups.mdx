## 关注组管理

关注组（Follow Groups）允许用户将不同的关注对象（如公司、账号等）组织成组，便于统一管理和获取相关内容。

---

## 创建关注组

**URL**: `/v1/following/groups`
**方法**: `POST`
**描述**: 创建一个新的关注组

### 请求参数

| 参数名         | 类型                   | 描述                                                                 |
|----------------|------------------------|----------------------------------------------------------------------|
| name           | str                   | 组名称                                                               |
| follow_values   | list[FollowValue]      | 关注项列表（可选），每个元素包含 `follow_type` 和 `follow_value` |

FollowValue 结构：

| 参数名         | 类型   | 描述                  |
|----------------|--------|-----------------------|
| follow_type    | int    | 关注类型：1=公司，3=频道 |
| follow_value   | str    | 关注值（如股票代码、频道 ID）   |

### 响应参数

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| group_id       | str   | 关注组 ID             |
| name           | str   | 组名称                 |
| count          | int   | 关注项数量               |

### 示例

**请求**:

```json
{
  "name": "My Stocks",
  "follow_values": [
    { "follow_type": 1, "follow_value": "US:AAPL" },
    { "follow_type": 1, "follow_value": "HK:00700" }
  ]
}
```

**响应**:

```json
{
  "group_id": "group_123",
  "name": "My Stocks",
  "count": 2
}
```

## 获取关注组列表

**URL**: `/v1/following/groups`
**方法**: `GET`
**描述**: 获取当前用户的所有关注组

### 请求参数

无需参数

### 响应参数

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| groups         | list  | 关注组列表               |

每个关注组包含：

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| group_id       | str   | 关注组 ID             |
| name           | str   | 组名称                 |
| count          | int   | 关注项数量               |

### 示例

**响应**:

```json
{
  "groups": [
    {
      "group_id": "group_123",
      "name": "My Stocks",
      "count": 2
    },
    {
      "group_id": "group_456",
      "name": "Tech News",
      "count": 5
    }
  ]
}
```

## 获取关注组文档

**URL**: `/v1/following/groups/{group_id}/docs`
**方法**: `GET`
**描述**: 获取关注组的文档列表

### 路径参数

| 参数名         | 类型   | 描述       |
|----------------|--------|-----------|
| group_id       | str    | 关注组 ID   |

### 查询参数

| 参数名         | 类型   | 描述                         |
|----------------|--------|----------------------------|
| page_num       | int    | 页码，默认值为 `1`              |
| page_size      | int    | 每页条目数，默认值为 `10`，最大值为 `100` |

### 响应参数

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| items          | list  | 文档列表               |
| total_count    | int   | 总文档数量              |
| page_num       | int   | 当前页码                |
| page_size      | int   | 每页条目数               |

每个文档包含（具体字段根据实际情况而定）：

| 参数名            | 类型    | 描述                 |
|----------------|-------|----------------------|
| id             | str   | 文档 ID               |
| title          | str   | 文档标题                |
| url            | str   | 文档链接                |
| ...            | ...   | 其他文档字段               |

### 示例

**响应**:

```json
{
  "items": [
    {
      "id": "doc_123",
      "title": "Sample Document Title",
      "url": "https://example.com/doc/123"
    }
  ],
  "total_count": 100,
  "page_num": 1,
  "page_size": 10
}
```
