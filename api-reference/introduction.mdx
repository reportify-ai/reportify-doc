## 基础信息

Reportify API 基于 REST 原则构建。我们在每个请求中强制执行 HTTPS，以提高数据安全性、完整性和隐私性。该 API 不支持 HTTP。

所有 API 请求的基础 URL 为：
```
https://api.reportify.cn
```

## 权限验证

要访问 Reportify 的 API，您需要在请求的 Header 中加入 `Authorization` 头信息，其中包含您的 Bearer Token：

```
Authorization: Bearer 447460****09c9
```
