## 股票价格

**URL**: `/v1/stock/company-prices`
**方法**: `POST`  
**描述**: 获取当前和历史股票价格数据。

### 请求参数
| 参数名      | 类型   | 必填 | 描述                                      |
|-------------|--------|------|-------------------------------------------|
| symbol      | str    | 是   | 股票代码，如 `HK:00700`、`US:AAPL`。      |
| start_date  | str    | 否   | 开始日期（YYYY-MM-DD）。                  |
| end_date    | str    | 否   | 结束日期（YYYY-MM-DD）。                  |

### 响应参数
| 参数名              | 类型    | 描述                                       |
|---------------------|---------|-------------------------------------------|
| status              | int     | HTTP 状态码。                              |
| code                | int     | 响应代码（0 表示成功）。                   |
| message             | str     | 响应消息。                                 |
| data                | object  | 价格数据。                                 |
| &nbsp;&nbsp;items   | array   | 价格列表。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;date                  | str     | 日期。                                     |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_price           | number  | 股票价格。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;stock_change_percent  | number  | 涨跌幅。                                   |
| &nbsp;&nbsp;&nbsp;&nbsp;trade_volume          | int     | 成交量。                                   |
| &nbsp;&nbsp;&nbsp;&nbsp;trade_amount          | number  | 成交额。                                   |
| &nbsp;&nbsp;&nbsp;&nbsp;market_capitalization | number  | 市值。                                     |
| &nbsp;&nbsp;&nbsp;&nbsp;pe_ttm                | number  | 市盈率（TTM）。                            |
| &nbsp;&nbsp;&nbsp;&nbsp;pb                    | number  | 市净率。                                   |

### 请求示例

**cURL**
```bash
curl -X POST https://api.reportify.cn/v1/stock/company-prices \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json" \
-d '{
    "symbol": "HK:00700",
    "start_date": "2025-12-01",
    "end_date": "2025-12-09"
}'
```

**Python**
```python
import requests

url = "https://api.reportify.cn/v1/stock/company-prices"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}
data = {
    "symbol": "HK:00700",
    "start_date": "2025-12-01",
    "end_date": "2025-12-09"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
