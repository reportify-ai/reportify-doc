## 公司股东

**URL**: `/v1/stock/company-shareholders`
**方法**: `POST`  
**描述**: 获取公司股东信息列表。

### 请求参数
| 参数名  | 类型   | 必填 | 描述                                                      |
|---------|--------|------|-----------------------------------------------------------|
| symbol  | str    | 是   | 股票代码，如 `HK:00700`、`US:AAPL`。                      |
| type    | str    | 否   | 类型：`shareholders` 或 `outstanding_shareholders`。      |
| limit   | int    | 否   | 返回数量限制，默认 10。                                   |

### 响应参数
| 参数名              | 类型    | 描述                                       |
|---------------------|---------|-------------------------------------------|
| status              | int     | HTTP 状态码。                              |
| code                | int     | 响应代码（0 表示成功）。                   |
| message             | str     | 响应消息。                                 |
| data                | object  | 股东数据。                                 |
| &nbsp;&nbsp;items   | array   | 股东列表。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;name     | str     | 股东名称。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;holdings | int     | 持股数量。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;percent  | number  | 持股比例。                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;date     | str     | 数据日期。                                 |

### 请求示例

**cURL**
```bash
curl -X POST https://api.reportify.cn/v1/stock/company-shareholders \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json" \
-d '{
    "symbol": "HK:00700",
    "limit": 5
}'
```

**Python**
```python
import requests

url = "https://api.reportify.cn/v1/stock/company-shareholders"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}
data = {
    "symbol": "HK:00700",
    "limit": 5
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
