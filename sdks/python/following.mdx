## 概述

Following 模块提供关注组的管理功能，包括创建、查询、更新和删除关注组，以及管理组内的关注项。

## create_follow_group()

创建一个新的关注组。

```python
result = client.following.create_follow_group(
    name="My Stocks",
    follow_values=[
        {"follow_type": 1, "follow_value": "US:AAPL"},
        {"follow_type": 1, "follow_value": "HK:00700"}
    ]
)
print(f"创建成功: {result['group_id']}")
```

<ResponseField name="name" type="str">
  组名称
</ResponseField>

<ResponseField name="follow_values" type="list[dict]">
  关注项列表，每个元素包含 `follow_type`（int）和 `follow_value`（str）
</ResponseField>

<ResponseField name="返回值" type="dict">
  包含 `group_id`, `name`, `count` 的字典
</ResponseField>

**API 路径**：`POST /v1/following/groups`

---

## get_follow_groups()

获取当前用户的所有关注组。

```python
groups = client.following.get_follow_groups()
for group in groups:
    print(f"{group['name']}: {group['count']} items")
```

<ResponseField name="返回值" type="list[dict]">
  关注组列表，每个元素包含 `group_id`, `name`, `count`
</ResponseField>

**API 路径**：`GET /v1/following/groups`

---

## get_follow_group()

根据 ID 获取一个关注组的详细信息。

```python
group = client.following.get_follow_group("group_123")
print(f"{group['name']}: {group['count']} items")
for fv in group.get('follow_values', []):
    print(f"  - Type {fv['follow_type']}: {fv['follow_value']}")
```

<ResponseField name="group_id" type="str">
  关注组 ID
</ResponseField>

<ResponseField name="返回值" type="dict">
  包含 `group_id`, `name`, `count`, `follow_values` 的字典
</ResponseField>

**API 路径**：`GET /v1/following/groups/{group_id}`

---

<ResponseField name="返回值" type="dict">

## add_follows_to_group()

向关注组添加关注项。

```python
result = client.following.add_follows_to_group(
    "group_123",
    follow_values=[{"follow_type": 1, "follow_value": "US:TSLA"}]
)
print(result.get('message', 'Added'))
```

<ResponseField name="group_id" type="str">
  关注组 ID
</ResponseField>

<ResponseField name="follow_values" type="list[dict]">
  要添加的关注项列表
</ResponseField>

<ResponseField name="返回值" type="dict">
  操作结果
</ResponseField>

**API 路径**：`POST /v1/following/groups/{group_id}/follows`

---

## remove_follows_from_group()

从关注组中移除关注项。

```python
result = client.following.remove_follows_from_group("group_123", ["AAPL", "MSFT"])
print(result.get('message', 'Removed'))
```

<ResponseField name="group_id" type="str">
  关注组 ID
</ResponseField>

<ResponseField name="follow_values" type="list[str]">
  要删除的关注值列表（如股票代码、频道 ID）
</ResponseField>

<ResponseField name="返回值" type="dict">
  操作结果
</ResponseField>

**API 路径**：`DELETE /v1/following/groups/{group_id}/follows`

---

## set_group_follows()

设置关注组的关注项（替换所有现有关注项）。

```python
result = client.following.set_group_follows(
    "group_123",
    follow_values=[
        {"follow_type": 1, "follow_value": "US:AAPL"},
        {"follow_type": 1, "follow_value": "HK:00700"}
    ]
)
print(result.get('message', 'Set'))
```

<ResponseField name="group_id" type="str">
  关注组 ID
</ResponseField>

<ResponseField name="follow_values" type="list[dict]">
  新的关注项列表
</ResponseField>

<ResponseField name="返回值" type="dict">
  操作结果
</ResponseField>

**API 路径**：`PUT /v1/following/groups/{group_id}/follows`

---

## get_follow_group_docs()

获取关注组的文档列表。

```python
result = client.following.get_follow_group_docs("group_123", page_num=1, page_size=20)
print(f"Total reports: {result['total_count']}")
for item in result['items']:
    print(f"  - {item.get('title', 'Untitled')}")
```

<ResponseField name="group_id" type="str">
  关注组 ID
</ResponseField>

<ResponseField name="page_num" type="int">
  页码，默认值为 `1`
</ResponseField>

<ResponseField name="page_size" type="int">
  每页条目数，默认值为 `10`
</ResponseField>

<ResponseField name="返回值" type="dict">
  包含 `items`, `total_count`, `page_num`, `page_size` 的字典
</ResponseField>

**API 路径**：`GET /v1/following/groups/{group_id}/docs`
