## 公司信息

### overview()

获取公司概览信息。

```python
info = client.stock.overview("US:AAPL")
print(info["name"], info["sector"], info["industry"])
```

<ParamField path="symbol" type="string" required>
  股票代码，格式：`市场:代码`，如 `US:AAPL`, `HK:0700`, `CN:600519`
</ParamField>

<ResponseField name="返回值" type="dict">
  公司信息，包含 `name`, `description`, `sector`, `industry`, `market_cap` 等
</ResponseField>

---

### shareholders()

获取主要股东列表。

```python
shareholders = client.stock.shareholders("US:AAPL")
for holder in shareholders:
    print(holder["name"], holder["percentage"])
```

<ParamField path="symbol" type="string" required>
  股票代码
</ParamField>

<ResponseField name="返回值" type="list[dict]">
  股东列表，包含 `name`, `shares`, `percentage` 等
</ResponseField>

## 财务报表

### income_statement()

获取利润表数据。

```python
income = client.stock.income_statement(
    symbol="US:AAPL",
    period="quarterly",
    limit=8
)
print(income[["revenue", "net_income", "eps"]].head())
```

<ParamField path="symbol" type="string" required>
  股票代码
</ParamField>

<ParamField path="period" type="string" default="annual">
  报告期间：`annual`（年报）或 `quarterly`（季报）
</ParamField>

<ParamField path="limit" type="int" default="10">
  返回期数
</ParamField>

<ResponseField name="返回值" type="DataFrame">
  利润表数据，以日期为索引，包含 `revenue`, `gross_profit`, `operating_income`, `net_income`, `eps` 等列
</ResponseField>

---

## 行情数据

### prices()

获取历史股价数据。

```python
prices = client.stock.prices(
    symbol="US:AAPL",
    start_date="2024-01-01",
    end_date="2024-06-30",
    limit=100
)
print(prices[["close", "volume", "market_cap"]].tail())
```

<ParamField path="symbol" type="string" required>
  股票代码
</ParamField>

<ParamField path="start_date" type="str">
  开始日期，格式：`YYYY-MM-DD`
</ParamField>

<ParamField path="end_date" type="str">
  结束日期，格式：`YYYY-MM-DD`
</ParamField>

<ParamField path="limit" type="int" default="100">
  返回记录数
</ParamField>

<ResponseField name="返回值" type="DataFrame">
  股价数据，包含 `close`, `volume`, `market_cap`, `pe`, `ps` 等
</ResponseField>

---

### kline()

获取 K 线数据。

```python
kline = client.stock.kline(
    symbol="US:TSLA",
    interval="1d",
    adjust="forward",
    limit=100
)
print(kline[["open", "high", "low", "close", "volume"]].tail())
```

<ParamField path="symbol" type="string" required>
  股票代码
</ParamField>

<ParamField path="interval" type="string" default="1d">
  时间周期：`1d`（日线）, `1w`（周线）, `1m`（月线）
</ParamField>

<ParamField path="adjust" type="string" default="forward">
  复权类型：`forward`（前复权）, `backward`（后复权）, `none`（不复权）
</ParamField>

<ParamField path="start_date" type="str">
  开始日期
</ParamField>

<ParamField path="end_date" type="str">
  结束日期
</ParamField>

<ParamField path="limit" type="int" default="100">
  返回记录数
</ParamField>

## 指数和行业

### index_constituents()

获取指数成分股列表。

```python
# 获取沪深300指数成分股
csi300 = client.stock.index_constituents("000300")
print(csi300[["market", "symbol"]])

# 获取创业板指数成分股
chinext = client.stock.index_constituents("399006")
```

<ParamField path="symbol" type="string" required>
  指数代码，如 `000300`（沪深300）, `000016`（上证50）, `399006`（创业板指）
</ParamField>

<ResponseField name="返回值" type="DataFrame">
  成分股列表，包含 `market`, `symbol` 等
</ResponseField>

---

### index_funds()

获取指数跟踪基金列表。

```python
# 获取沪深300指数跟踪基金
funds = client.stock.index_funds("000300")
print(funds[["symbol", "short_name", "name"]])

# 获取中证500指数跟踪基金
funds = client.stock.index_funds("000905")
```

<ParamField path="symbol" type="string" required>
  指数代码，如 `000300`（沪深300）, `000016`（上证50）, `000905`（中证500）
</ParamField>

<ResponseField name="返回值" type="DataFrame">
  跟踪基金列表，包含 `symbol`, `short_name`, `name` 等
</ResponseField>

---

### industry_constituents()

获取行业成分股列表。

```python
# 获取中国军工行业成分股
stocks = client.stock.industry_constituents("cn", "军工")
print(stocks[["symbol", "name", "industry_one_level_name"]])

# 使用特定分类标准
stocks = client.stock.industry_constituents("cn", "军工", type="sw")
```

<ParamField path="market" type="string" required>
  股票市场：`cn`（中国）, `hk`（香港）, `us`（美国）
</ParamField>

<ParamField path="name" type="string" required>
  行业名称，如 `军工`, `Technology`
</ParamField>

<ParamField path="type" type="string">
  行业分类类型：
  - CN: `sw`（申万，默认）, `wind`
  - HK: `hs`（恒生，默认）
  - US: `GICS`（默认）
</ParamField>

<ResponseField name="返回值" type="DataFrame">
  成分股列表，包含 `symbol`, `name`, `industry_one_level_name`, `industry_second_level_name`, `industry_third_level_name` 等
</ResponseField>

## 选股和日历

### screener()

股票筛选器。

```python
stocks = client.stock.screener(
    market="US",
    min_market_cap=1e10,  # 100亿美元以上
    max_pe=30,
    limit=50
)
```

<ParamField path="market" type="str">
  市场：`US`, `HK`, `CN`
</ParamField>

<ParamField path="sector" type="str">
  行业板块
</ParamField>

<ParamField path="min_market_cap" type="float">
  最小市值
</ParamField>

<ParamField path="max_market_cap" type="float">
  最大市值
</ParamField>

<ParamField path="min_pe" type="float">
  最小市盈率
</ParamField>

<ParamField path="max_pe" type="float">
  最大市盈率
</ParamField>

<ParamField path="limit" type="int" default="50">
  返回数量
</ParamField>
