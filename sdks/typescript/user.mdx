## 概述

用户数据模块提供用户个人数据的访问，包括关注的公司列表等。

## followedCompanies()

获取用户关注的所有公司列表。

```typescript
const companies = await client.user.followedCompanies();
companies.forEach(company => {
  console.log(company.name, company.symbol);
});
```

**返回值：** `Promise<FollowedCompany[]>` - 关注公司列表

---

## 类型定义

```typescript
interface FollowedCompany {
  symbol: string;       // 股票代码，如 US:AAPL
  ticker: string;       // 股票简码，如 AAPL
  market: string;       // 市场，如 US, HK, CN
  name: string;         // 公司名称
  chineseName: string;  // 中文名称
  englishName: string;  // 英文名称
  logo: string;         // 公司 Logo URL
  followedAt: number;   // 关注时间戳（毫秒）
}
```

## 使用场景

```typescript
// 获取关注公司并查询其最新动态
const companies = await client.user.followedCompanies();
console.log(`共关注 ${companies.length} 家公司`);

companies.slice(0, 5).forEach(company => {
  console.log(`- ${company.name} (${company.symbol})`);
});

// 结合时间线使用
const timeline = await client.timeline.companies({ num: 10 });
timeline.forEach(item => {
  console.log(`- ${item.title}`);
});
```

## createFollowGroup()

创建一个新的关注组。

```typescript
// 创建关注组并添加关注项
const result = await client.user.createFollowGroup('我的自选股', [
  { follow_type: 1, follow_value: 'US:AAPL' },
  { follow_type: 1, follow_value: 'HK:00700' }
]);
console.log(`创建成功: ${result.group_id}`);
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | 是 | 关注组名称 |
| `followValues` | `Array<{ follow_type: number; follow_value: string }>` | 否 | 关注项列表 |

**返回值：** `Promise<{ group_id: string; name: string; count: number }>`

## getFollowGroups()

获取用户的所有关注组列表。

```typescript
const groups = await client.user.getFollowGroups();
groups.forEach(group => {
  console.log(`${group.name}: ${group.count} 个关注项`);
});
```

**返回值：** `Promise<Array<{ group_id: string; name: string; count: number }>>`

---

## getFollowGroup()

获取单个关注组的详细信息，包括关注项列表。

```typescript
const group = await client.user.getFollowGroup('group_123');
console.log(`${group.name}: ${group.count} 个关注项`);

group.follow_values.forEach(fv => {
  console.log(`  - 类型 ${fv.follow_type}: ${fv.follow_value}`);
});
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `groupId` | `string` | 是 | 关注组 ID |

**返回值：** `Promise<{ group_id: string; name: string; count: number; follow_values: Array<{ follow_type: number; follow_value: string }> }>`

**返回值：** `Promise<{ group_id: string; name: string; count: number }>`

## addFollowsToGroup()

向关注组添加关注项。

```typescript
await client.user.addFollowsToGroup('group_123', [
  { follow_type: 1, follow_value: 'US:TSLA' }
]);
console.log('添加成功');
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `groupId` | `string` | 是 | 关注组 ID |
| `followValues` | `Array<{ follow_type: number; follow_value: string }>` | 是 | 要添加的关注项列表 |

---

## removeFollowsFromGroup()

从关注组删除关注项（通过 follow_value）。

```typescript
await client.user.removeFollowsFromGroup('group_123', ['US:AAPL', 'HK:00700']);
console.log('删除成功');
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `groupId` | `string` | 是 | 关注组 ID |
| `followValues` | `string[]` | 是 | 要删除的关注值列表（如股票代码、频道 ID） |

## setGroupFollows()

设置关注组的所有关注项（替换所有现有项）。

```typescript
await client.user.setGroupFollows('group_123', [
  { follow_type: 1, follow_value: 'US:AAPL' },
  { follow_type: 1, follow_value: 'HK:00700' }
]);
console.log('设置成功');
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `groupId` | `string` | 是 | 关注组 ID |
| `followValues` | `Array<{ follow_type: number; follow_value: string }>` | 是 | 新的关注项列表 |

## getGroupReports()

获取关注组的报告列表。

```typescript
const result = await client.user.getGroupReports('group_123', 1, 20);
console.log(`共 ${result.total_count} 条报告`);

result.items.forEach(report => {
  console.log(`- ${report.title || '未命名'}`);
});
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `groupId` | `string` | 是 | 关注组 ID |
| `pageNum` | `number` | 否 | 页码，默认 1 |
| `pageSize` | `number` | 否 | 每页数量，默认 10 |

**返回值：** `Promise<{ items: any[]; total_count: number; page_num: number; page_size: number }>`
