##  Reportify 是什么

Reportify 是一个金融投资智能体平台，可以用自然语言构建智能体与自动化流程，结合全球多市场的金融数据与可扩展应用接入，为金融投资的分析、操作与协作提供统一且高效的智能化方案。

使用方式：
浏览器直接访问 https://reportify.cn  

Reportify 亮点功能：
* 可通过自然语言自定义 Agent，更精确控制 Agent 内容和行为
* 可通过自然语言自定 Agent 任务，后台自动运行发送邮件提醒
* 整合众多第三方应用，还可以自定义 MCP 丰富 Agent 能力
* 内置众多官方 Agent 和海量用户分享 Agent，开箱即用
* 支持 A 股、港股、美股全市场财报、电话会议、新闻、研报内容聚合，
* 针对各种文档自动进行总结快速浏览文档核心内容和关键要点
* 每一个总结和关键要点都有引用出处并且对原文内容进行高亮，方便回溯原文
* 支持财报、电话会、新闻超长文档全文中英文对照翻译，以及选中文本定向翻译
* 支持历史问答会话保存，方便回溯和分享，构建投资研究知识库
* 自定义文件上传，多文件问答，最大支持单文件 1GB
* 通过标签和任意搜索条件订阅最新相关文档实时更新

## 文档概述
Reportify API 提供了功能强大的接口，旨在帮助用户通过 API 的方式访问 Reporitfy 的服务。通过这些 API，开发者可以实现文件搜索、上传、问答等功能，并高效地获取和管理文档内容。本 API 文档详细介绍了各个接口的请求方法、参数要求、响应格式和示例，帮助开发者快速集成 Reportify 的功能。

## 示例应用场景
- **金融文档聚合**：将多个机构发布的报告集中存储和管理，便于跨公司、跨行业的分析。
- **智能内容检索和问答**：通过语义搜索和问答，快速定位特定领域或事件的关键信息，支持精准决策。
- **报告生成与总结**：利用文档内容与总结功能，为研究员、分析师提供简洁、直接的文档摘要，提升工作效率。

## 快速入门
在开始使用 API 前，请确保获取到一个有效的 API 密钥，并在请求的 `Authorization` 头中添加 `Bearer {API_KEY}` 以进行身份验证。
每个接口的请求示例和响应示例都采用 JSON 格式，开发者可以使用常见的 HTTP 客户端（如 `cURL`、`requests`）进行 API 测试与集成。
