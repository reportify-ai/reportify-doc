##  Reportify 是什么

Reportify 是一个金融投资智能体平台，可以用自然语言构建智能体与自动化流程，结合全球多市场的金融数据与可扩展应用接入，为金融投资的分析、操作与协作提供统一且高效的智能化方案。

<Info>
  浏览器直接访问 [https://reportify.cn](https://reportify.cn) 开始使用
</Info>

## API 文档概述

Reportify API 提供了功能强大的接口，旨在帮助用户通过 API 的方式访问 Reportify 的服务。通过这些 API，开发者可以实现文件搜索、上传、问答等功能，并高效地获取和管理文档内容。本 API 文档详细介绍了各个接口的请求方法、参数要求、响应格式和示例，帮助开发者快速集成 Reportify 的功能。

## 快速开始

<Steps>
  <Step title="获取 API 密钥">
    登录 [Reportify 开发者中心](https://reportify.cn/developer) 获取您的 API Key
    
    <Info>
      请妥善保管您的 API Key，不要在公开场所泄露
    </Info>
  </Step>
  
  <Step title="配置认证">
    在所有 API 请求的 `Authorization` 头中添加 Bearer Token
    
    <CodeGroup>
    ```bash cURL
    curl -X POST "https://api.reportify.cn/v1/search" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json"
    ```
    
    ```python Python
    import requests

    headers = {
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    }
    
    response = requests.post(
        "https://api.reportify.cn/v1/search",
        headers=headers
    )
    ```
    
    ```javascript JavaScript
    const headers = {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    };

    fetch('https://api.reportify.cn/v1/search', {
      method: 'POST',
      headers: headers
    });
    ```
    </CodeGroup>
  </Step>
  
  <Step title="开始调用">
    查看下方的 API 文档，选择您需要的接口开始使用
  </Step>
</Steps>
